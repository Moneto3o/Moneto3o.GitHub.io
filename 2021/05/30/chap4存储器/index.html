4.4. 说明存取周期和存取时间的区别。

答: 存取周期和存取时间的主要区别是：

存取时间仅为完成一次操作的时间，

而存取周期不仅包含操作时间，还包含操作后线路的恢复时间。

即：存取周期= 存取时间 + 恢复时间

4.7 一个容量为16K×32位的存储器，其地址线和数据线的总和是多少？当选用下列不同规格的存储芯片时，各需要多少片？

1K×4位，2K×8位，4K×4位，16K×1位，4K×8位，8K×8位

解: 地址线和数据线的总和 = 14 + 32 = 46根；

需要的片数为：
1K×4：16K×32 / 1K×4 = 16×8 = 128片
2K×8：16K×32 / 2K×8 = 8×4 = 32片
4K×4：16K×32 / 4K×4 = 4×8 = 32片
16K×1：16K×32 / 16K×1 = 32片
4K×8：16K×32 / 4K×8 = 4×4 = 16片
8K×8：16K×32 / 8K×8 = 2×4 = 8片

例1. 16K×8位的DRAM芯片，设存储器读/写周期（存取周期）为0.5µs,刷新周期为2ms 。CPU在1µs 内至少要访存一次。试问采用哪种刷新方式比较合理？刷新信号周期是多少？对全部存储单元刷新一遍所需的实际刷新时间是多少？

解: 根据题目要求，不能使用集中式刷新，分散式可以但不合理，故采用异步式刷新。

设16K×8位存储芯片的阵列结构为128行×128 ×8列，按行刷新，
则异步刷新的刷新信号周期为：
2ms / 128 = 15.6 µs
对全部存储单元刷新一遍所需的实际时间为:
0.5×128=64 (µs)

（1）采用集中刷新方式刷新间隔为 ：2ms ，其中刷新死时间为： 256× 0.1 μs=25.6 μs
（2）采用分散刷新方式刷新间隔为： 256×（ 0.1 μs+×0.1 μs）=51.2 μs
（3）采用异步刷新方式刷新间隔为：:2ms

4.29 假设CPU执行某段程序时共访问Cache命中4800次，访问主存200次，已知Cache的存取周期为30ns，主存的存取周期为150ns，求Cache的命中率以及Cache-主存系统的平均访问时间和效率，试问该系统的性能提高了多少倍？

Cache被访问命中率为：4800/(4800+200)=24/25=96%

则Cache-主存系统的平均访问时间为：ta=0.96* 30ns+(1-0.96)* 150ns=34.8ns

Cache-主存系统的访问效率为：e=tc/ta* 100%=30/34.8* 100%=86.2%

性能为原来的150ns/34.8ns=4.31倍，即提高了3.31倍。

4.31 设主存容量为1MB，采用直接映射方式的Cache容量为16KB，块长为4个字，每字32 位。试问主存地址为ABCDEH的存储单元在Cache中的什么位置？

解： 主存按字节编址，所以Cache地址及映射均按字节处理

Cache容量16KB=2^14B，地址共格式为14位，分为16KB/(4* 32/8B)=2^10块，每块4*32/8=16B=2^4B，Cache地址格式为：

Cache字块地址（10位） 字块内地址（4位）

主存容量1MB=2^20B，地址共格式为20位，分为1MB/(4*32/8B)=2^16块，每块2^4B，采用直接映射方式，主存字块标记为20-14=6位，主存地址格式为：

主存字块标记（6位） Cache字块地址（10位） 字块内地址（4位）

主存地址为ABCDEH=1010 1011 1100 1101 1110B，

故该主存单元应映射到Cache的101010块的第1110字节，即第42块第14字节位置。

或者在Cache的第11 1100 1101 1110=3CDEH字节位置。

例4 一个组相联地址映像Cache由64个存储块组成，每组包含4个存储块。主存包含4096个存储块，每块由8个字组成，每字32位。存储器按字节编址，访存地址为字地址。
(1)写出Cache和主存的地址位数和地址格式
(2)块表容量为多大？

(3)主存地址18AB9H映像到Cache的哪一块

解：主存按字节编址，所以Cache地址及映射均按字节处理

(1) Cache容量=64* 8* 4B=2048B=2^11B —— 11位地址

每组包含4个存储块，所以组内块号为2位;Cache有64/4=2^4个组，所以组号为4位。

Cache字块地址（6位） 字块内地址（5位）

主存容量=4096* 8* 4B=128KB=2^17B—— 17位地址

主存字块标记（8位） 组地址（4位） 字块内地址（5位）

(2)块表容量：由于cache块号为6位，因此有2的6次方个单元数量，由于每个单元中存放的是块标记，加上标记有效位，块表容量为：
64字* (8+1)位，即 64字* 9位

(3) 主存地址 0001 1000 1010 1011 1001

（2）设Cache的初态为空，CPU依次从主存第0、1、2……89号单元读出90个字（主存一次读出一个字），并重复按此次序读8次，问命中率是多少？

主存字块标记（10位） 组地址（7位） 字块内地址（5位）

(2) 未命中次数 90 / 8 ≈12
命中率 h=(90* 8-12) / 90* 8 = 98.3%

(3) 没有Cache的访问时间为6t* 720，
有Cache的访问时间为 t* (720-12)+6t* 12，
则有Cache和没有Cache相比，速度提高倍数:

​ 或: ta = h × tc+ (1－h)× tm
​ = 0.983t +(1-0.983) 6t = 1.0905t

例6 假设CPU执行某段程序时共访问Cache命中4800次，访问主存200次，已知Cache的存取周期是30ns，主存的存取周期是150ns，求Cache命中率、cache-主存系统的平均访问时间和效率，试问该系统的性能提高了多少？

​ e = tc / ta = 30 /34.8 = 86.2%
