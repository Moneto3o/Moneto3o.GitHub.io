R语言复习

概述

  1. R语言是一种解释型语言

  2. 功能: 绘图功能、数据分析

    功能的实现手段：工具（包、库）、编程语言

  3. 工作空间

    获得工作空间：getwd()

    设置工作空间：setwd(‘路径’)

    工作空间中查看文件：list.files() dir()

  4. R包

    获取某个包的功能: help(package = “XX”)

    安装R包：install.packages(‘car’)

    加载: library(car)

    移除R包: remove.package(‘car’)

入门

  1. 变量定义：英文字母+数字+下划线+英文点，

    开头：英文字母+英文点（后续不能直接跟数字）

  2. 变量赋值: = <- <<-

  3. 查询变量: ls()

    查询所有变量的详细信息：ls.str()

    查询某个变量的详细信息：str(name)

    查询隐藏变量：ls(all.names = TRUE)

    条件查询：ls(pattern = “”)

  4. 删除变量

    rm(name)—-删除一个变量

    rm(name1,name2,…)—–删除列举出来的所有变量

    rm(list = ls())—-删除所有变量

  5. 工作历史: history()

  6. 保存: save.image() 存储为RData文件(只保存命令 数据 不存图片)

  7. 退出: q()

  8. 注释: #
